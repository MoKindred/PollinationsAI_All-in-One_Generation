# Pollinations AI 全能生成工具

一个功能强大的 Pollinations AI 一站式生成工具，支持文本、图片、视频、音频的 AI 生成。

## 功能特点

### 🎯 核心功能
- **文本生成**：支持多种大语言模型，包括 openai、deepseek 等
- **图片生成**：支持 Flux、zimage、gptimage、grok-imagine 等高质量图片模型
- **视频生成**：支持 Veo、Seedance、Grok Video 等视频模型
- **音频生成**：支持 suno、whisper 等音频模型

### 🌍 多语言支持
- 中文界面
- 英文界面
- 自动检测用户浏览器语言
- 实时语言切换

### 🎨 用户界面
- 现代化玻璃态设计
- 响应式布局，支持移动端
- 左右分栏设计，操作便捷
- 实时生成状态反馈
- 模型元数据详细展示

### ⚙️ 高级功能
- **文本参数**：温度值、最大长度、角色选择
- **图片参数**：尺寸、风格、负面提示词、质量、种子值、增强、透明背景
- **视频参数**：时长、分辨率、帧率、宽高比、音频
- **音频参数**：时长、语音、格式、语速、纯音乐
